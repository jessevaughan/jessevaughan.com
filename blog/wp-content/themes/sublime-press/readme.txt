== Description ==
Sublime is a flexible responsive blog theme. It is designed to be simple, clean and clear and uses a responsive layout so that it looks beautiful on desktops, tablet PCs and smart phones.

=== Features ===
Responsive Layout, Page Navigation, Search Bar, Theme Customiser.

=== Widgets Areas ===
The Theme has customisable sidebar and footer areas.
You can use these area to customise the content of your website.
